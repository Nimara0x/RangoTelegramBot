# Rango Telegram Bot 


### Install dependencies:

```bash
pip install -r requirements.txt
```

### Set up `.env`:

```dotenv
TOKEN='xxxx'  # your bot token here
RANGO_API_KEY='YourRangoApi'
RANGO_BASE_URL='https://api.rango.exchange/'
DEVELOPMENT='true' # For testing
```

### Run bot

```python
python3 src/main.py
```
